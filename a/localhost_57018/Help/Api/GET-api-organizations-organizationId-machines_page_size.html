GET api/organizations/{organizationId}/machines?page={page}&size={size}

Get a list of machines belonging to an organization.

Request Information

URI Parameters

NameDescriptionTypeAdditional information
organizationId

The ID of the Organization whose ApiMachines you'd like to retrieve.

 integer

Required
page

The page of records you want to get.

 integer

Default value is 1
size

The number of records per page retrieved.

 integer

Default value is 50

Body Parameters

None.

Response Information

Resource Description

A list of ApiMachines, or the errors that occured during their retrieval.

Collection of ApiMachine
NameDescriptionTypeAdditional information
Id

Machine ID

 integer

None.
SiteId

ID of the Site this machine belongs to

 integer

Required
SerialNumber

Machine PROSE ID

 string

String length: inclusive between 0 and 15
Name

Friendly Name of the machine

 string

Required

String length: inclusive between 1 and 50
Location

Descriptive location of the machine

 string

Required

String length: inclusive between 1 and 50
PlanogramIds

A list containing the ID of the child Planograms.

 Collection of integer

None.
