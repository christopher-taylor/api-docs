GET api/organizations?page={page}&size={size}

Return a list of organizations the user has access to.

Request Information

URI Parameters

NameDescriptionTypeAdditional information
page

The page of records you would like to retrieve.

 integer

Default value is 1
size

The number of records per page.

 integer

Default value is 50

Body Parameters

None.

Response Information

Resource Description

A list of all organizations the requester has access to.

Collection of ApiOrganization
