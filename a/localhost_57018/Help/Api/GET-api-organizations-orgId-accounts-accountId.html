GET api/organizations/{orgId}/accounts/{accountId}

Retrieve an account by ID from an organization.

Request Information

URI Parameters

NameDescriptionTypeAdditional information
orgId

The ID of the organization whose accounts you'd like to retrieve.

 integer

Required
accountId

The ID of the account you'd like to retrieve.

 integer

Required

Body Parameters

None.

Response Information

The requested account (if it exists).
